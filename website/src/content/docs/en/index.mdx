## Why not just hack Claude Code's env vars

```diff
- Model thinks it's Claude ("I am Claude" while talking to MiMo)
- All error messages use Anthropic terminology
- Config conflicts (Claude one moment, MiMo the next — opening both = chaos)
- MiMo's thinking field isn't shown
- Prompt cache stats aren't visible
- Violates Claude Code's ToS (an Anthropic tool calling a non-Claude model)
+ Native MiMo experience · model identifies itself correctly
+ Every UI / command / log is MiMo-flavored
+ Isolated `~/.mimo/` · coexists with Claude Code, never collides
+ Live thinking + cache-hit rendering
+ Fully compliant: original + uses MiMo's official public protocol
```

## Free forever · No Pro / Enterprise tier

mimo-tui is MIT licensed. **No paid tier, no data resale, no ads**.

Why?
- The tool is for MiMo Token Plan users — they already paid Xiaomi
- We want MiMo's ecosystem to have a Claude-Code-quality CLI
