## 5-second pitch

A terminal AI coding tool exactly like [Claude Code](https://docs.claude.com/en/docs/claude-code), [Codex CLI](https://github.com/openai/codex), and [Gemini CLI](https://github.com/google-gemini/gemini-cli) — except powered by **Xiaomi's MiMo model**.

```bash
npm install -g mimo-tui
mimo                     # First run → guided setup with your own API key
cd ~/code/my-project
mimo

> Refactor Header.tsx into a function component
> Run npm test and fix all errors
> Write a README for this project
```

## Install

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install -g mimo-tui
    ```
  </TabItem>
  <TabItem label="Homebrew">
    ```bash
    brew install mimo-tui/mimo-tui/mimo-tui
    ```
  </TabItem>
  <TabItem label="Cargo">
    ```bash
    cargo install mimo-tui
    ```
  </TabItem>
  <TabItem label="curl">
    ```bash
    curl -fsSL https://mimo-tui.pages.dev/install.sh | sh
    ```
  </TabItem>
  <TabItem label="Docker">
    ```bash
    docker run -it ghcr.io/duolaamengweb3/mimo-tui:latest
    ```
  </TabItem>
</Tabs>
