## 5 秒看懂

跟 [Claude Code](https://docs.claude.com/en/docs/claude-code)、[Codex CLI](https://github.com/openai/codex)、[Gemini CLI](https://github.com/google-gemini/gemini-cli) 一样的终端 AI 编程工具，区别是底层用**小米 MiMo 模型**。

```bash
npm install -g mimo-tui  # 装
mimo                     # 首次启动 → 引导你配自己的 API key
cd ~/code/my-project     # 进项目
mimo                     # 开始用

```

## 为什么用 mimo-tui 不用 Claude Code

<CardGrid>
  <Card title="省钱" icon="seti:powershell">
    MiMo Token Plan 比 Claude Opus 便宜得多，1M 上下文还省 40-60% token
  </Card>
  <Card title="原生体验" icon="approve-check">
    不用配 Claude Code 然后改环境变量套壳，装完即用
  </Card>
  <Card title="看得见用量" icon="bars">
    实时知道套餐还剩多少 token、本次缓存命中省了多少
  </Card>
  <Card title="看得见 thinking" icon="open-book">
    MiMo 推理过程实时渲染、可折叠展开
  </Card>
  <Card title="配置独立" icon="setting">
    `~/.mimo/` vs `~/.claude/`，跟 Claude Code 并存互不污染
  </Card>
  <Card title="完全开源不收费" icon="heart">
    MIT 许可，没有 Pro / 企业版，永远免费
  </Card>
</CardGrid>
