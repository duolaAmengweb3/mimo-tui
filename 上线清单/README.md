## 工作方法建议

1. **从 04 开始**——这是关键路径，别的都在等它
2. **并行推 05/06**——官网和文档可以一边开发一边写
3. **每周末做一次 mini 自检**——避免最后一周才发现东西没做
4. **不要因为某项还没做完拖延整体发布**——所有项都要 100% 才发，不要"差不多就行"
