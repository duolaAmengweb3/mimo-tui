# 01 · 品牌视觉

> 目标：所有视觉资产准备齐全，可用于官网、README、社交媒体、应用商店
> 估时：1–2 天
> 依赖：无

## 1.1 Logo 定稿

- [ ] 从 `品牌/preview.html` 三方案选定 1 个（A / B / C）
- [ ] 删掉未选用的 SVG 文件
- [ ] 选定方案生成完整变体：
  - [ ] `logo.svg`（主版本，浅色背景用）
  - [ ] `logo-dark.svg`（深色背景用，颜色反向）
  - [ ] `logo-mono.svg`（单色版本，用于黑白印刷）
  - [ ] `mark.svg`（仅图标，不含 wordmark）
  - [ ] `wordmark.svg`（仅文字，不含图标）

## 1.2 Favicon 全套

- [ ] `favicon.ico`（16/32/48 多尺寸）
- [ ] `favicon-16x16.png`
- [ ] `favicon-32x32.png`
- [ ] `apple-touch-icon.png`（180x180）
- [ ] `android-chrome-192x192.png`
- [ ] `android-chrome-512x512.png`
- [ ] `site.webmanifest`

> 工具：用 [realfavicongenerator.net](https://realfavicongenerator.net) 一键生成

## 1.3 社交媒体卡片

- [ ] OG 图（1200x630）—— 首页通用
- [ ] OG 图 - 文档版（1200x630，含 "Docs" 标识）
- [ ] OG 图 - 博客版（1200x630，含标题占位）
- [ ] Twitter / X card 图（1200x675）
- [ ] GitHub 仓库社交预览图（1280x640）

## 1.4 应用商店 / 渠道图

- [ ] npm 包 logo（128x128）
- [ ] Docker Hub logo（256x256）
- [ ] Homebrew tap 头图（可选）

## 1.5 色彩规范文档

- [ ] 写 `品牌/colors.md`：
  - 主色 + HEX / RGB / HSL
  - 辅助色
  - 中性色（灰阶 9 档）
  - 暗色模式映射
  - 终端色（如果有 ANSI 输出场景）

## 1.6 字体规范

- [ ] 写 `品牌/typography.md`：
  - 主字体（等宽：JetBrains Mono / Geist Mono）
  - 副字体（无衬线：Inter / Geist Sans）
  - 字号阶梯
  - 行高规则

## 1.7 ASCII Logo（TUI 启动屏用）

- [ ] 在 `品牌/ascii-logo.txt` 设计启动时的 ASCII art
  - 示例（方案 A 风格）：
    ```
    ▎> mimo-tui
    ```
  - 单色 + 彩色两版

## 验收

- [ ] 所有资产用 `品牌/` 目录统一管理
- [ ] 浏览器打开 favicon 测试三个尺寸都清晰
- [ ] OG 图在 [opengraph.xyz](https://www.opengraph.xyz) 预览效果 OK
