# 03 · 代码仓库与协议

> 目标：仓库根目录所有"开源项目必备"文件齐全
> 估时：0.5 天
> 依赖：02 基础设施

## 3.1 必备根文件

- [ ] `README.md`（中英双语 / hero + 安装 + 快速开始 + 链接到官网）
- [ ] `README.zh-CN.md`（如果不放在同一文件）
- [ ] `LICENSE`（MIT）
- [ ] `LICENSE.zh-CN`（中文 MIT，可选）
- [ ] `CHANGELOG.md`（Keep a Changelog 格式）
- [ ] `CONTRIBUTING.md`（如何贡献 / 开发环境 / 提 PR 流程）
- [ ] `CODE_OF_CONDUCT.md`（Contributor Covenant 模板）
- [ ] `SECURITY.md`（安全问题报告流程）
- [ ] `.gitignore`（Rust + Node + macOS + JetBrains + VSCode）
- [ ] `.editorconfig`（统一格式）

## 3.2 .github/ 目录

- [ ] `.github/ISSUE_TEMPLATE/bug_report.yml`
- [ ] `.github/ISSUE_TEMPLATE/feature_request.yml`
- [ ] `.github/ISSUE_TEMPLATE/question.yml`
- [ ] `.github/ISSUE_TEMPLATE/config.yml`（指向 Discord 等）
- [ ] `.github/PULL_REQUEST_TEMPLATE.md`
- [ ] `.github/FUNDING.yml`（虽不收费，但可选放 GitHub Sponsors）
- [ ] `.github/dependabot.yml`（自动依赖更新）
- [ ] `.github/CODEOWNERS`

## 3.3 仓库 Settings

- [ ] About 卡片：写清楚 description / website / topics
- [ ] Topics：`mimo` `xiaomi` `tui` `cli` `coding-agent` `ai` `rust` `claude-code-alternative`
- [ ] Settings → Discussions 启用
- [ ] Settings → Branch protection（main 分支）：
  - [ ] 要求 PR review
  - [ ] 要求 CI 通过
  - [ ] 禁止 force push
- [ ] Settings → Pull Requests：默认 squash merge
