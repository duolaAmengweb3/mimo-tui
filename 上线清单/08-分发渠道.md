# 08 · 分发渠道

> 目标：用户在任何平台都能用最熟悉的方式装上
> 估时：2–3 天
> 依赖：04 主项目（有可发布二进制）/ 07 CI/CD

## 8.1 GitHub Releases（基础）

- [ ] tag → release 流程跑通（07 已配）
- [ ] 每平台二进制 + tar.gz/zip + sha256
- [ ] Release notes 从 CHANGELOG 自动生成
- [ ] Latest release 永远是 stable

## 8.2 npm

- [ ] `mimo-tui` 包名占位（02 已做）
- [ ] `npm/package.json` 配置：
  - [ ] bin: `mimo`
  - [ ] postinstall：根据 process.platform / arch 下载对应二进制
  - [ ] 用 `binary-install` 或 `node-install-binary` 库
- [ ] 测试：`npm install -g mimo-tui` 在三平台都能装
- [ ] 包描述 + keywords + repo + license
- [ ] README.md 用 npm 友好格式

## 8.3 Homebrew

- [ ] `homebrew-mimo-tui` tap 仓库（02 已建）
- [ ] `Formula/mimo-tui.rb`：
  ```ruby
  class MimoTui < Formula
    desc "Native open-source terminal AI coding agent for Xiaomi MiMo"
    homepage "https://mimo-tui.pages.dev"
    url "..."
    sha256 "..."
    license "MIT"
    def install
      bin.install "mimo"
    end
  end
  ```
- [ ] 测试：`brew tap mimo-tui/mimo-tui && brew install mimo-tui`
- [ ] CI 自动 bump（07 已配）

## 8.4 Cargo

- [ ] `cargo publish` workspace 各 crate
- [ ] 测试：`cargo install mimo-tui`
- [ ] crates.io 描述 + keywords + categories

## 8.5 Scoop（Windows）

- [ ] 创建 `scoop-mimo-tui` bucket 仓库
- [ ] `bucket/mimo-tui.json` manifest
- [ ] 测试：`scoop bucket add mimo-tui && scoop install mimo-tui`
- [ ] CI 自动 bump

## 8.6 Docker / GHCR

- [ ] `Dockerfile`（多 stage 构建，最终镜像 < 100MB）
- [ ] 多架构镜像（amd64 + arm64）
- [ ] tag 策略（latest / v1 / v1.0 / v1.0.0）
- [ ] 测试：`docker run ghcr.io/mimo-tui/mimo-tui:latest`
- [ ] Docker Hub 可选 mirror（如有需要）

## 8.7 一键安装脚本

- [ ] `install.sh` 写在官网 `public/install.sh`
- [ ] 功能：
  - [ ] 检测平台 / 架构
  - [ ] 下载 latest 二进制
  - [ ] 校验 sha256
  - [ ] 装到 PATH（/usr/local/bin 优先，~/.local/bin fallback）
  - [ ] 报告安装位置
- [ ] PowerShell 版本（Windows）`install.ps1`
- [ ] 测试：`curl -fsSL https://mimo-tui.pages.dev/install.sh | sh`
