能力

v0.3 实打实交付的部分

完整 agent loop

Anthropic Messages 协议 · 流式 SSE · 最多 32 步工具迭代 · 自动 context 压缩。

8 个内置工具

read · write · edit · shell · glob · grep · web_fetch · todo — 不靠 MCP 就能干活。

子 agent 并行

task 工具并行最多 10 个子 agent，把长任务拆成可管理的执行单元。

MCP 双协议

stdio + Streamable HTTP 都支持，官方 MCP server 装上就能用。

Skills 兼容

完全兼容 Anthropic Skills 格式，你的 `.claude/skills/*.md` 可以原样迁过来。

三平台沙箱

macOS Seatbelt · Linux Landlock · Windows Job Objects —— 默认开启。

智能区域路由

CN / SGP / AMS 三集群按延迟自动挑，区域 key 失效自动 fallback。

三种模式

Plan / Agent / Auto — 对应只读、需要审批、全自动三种风险偏好。

怎么用

三步进入工作状态

1

安装

npm / brew / cargo / docker / curl 一键脚本任选一种，30 秒内完成。

npm install -g mimo-tui
2

配 API key

带着你的 MiMo Token Plan key 跑一次 wizard，它会自动测三个 region 选最快的。

mimo init
3

开始写代码

进 TUI 直接对话，读写文件、跑 shell、拆子任务，全部在终端里完成。

mimo

快速安装

五种渠道，按你最顺手的来

npm（最常用）

npm install -g mimo-tui

Homebrew

brew install duolaAmengweb3/mimo-tui/mimo-tui

Cargo（Rust 用户）

cargo install --git https://github.com/duolaAmengweb3/mimo-tui mimo-tui

一键脚本

curl -fsSL https://mimo-tui.pages.dev/install.sh | sh

Docker

docker run -it -v "$PWD":/workspace ghcr.io/duolaamengweb3/mimo-tui:latest

装完后跑 mimo init 配一次 API key，下次直接 mimo 就能进 TUI。

关于 mimo-tui

mimo-tui 是一个独立的社区开源项目，目标是给 MiMo 模型生态补上一个 Claude Code / Codex CLI 水准的终端 agent。Rust 从零写，MIT 协议，不收费、不绑定托管、不二次代理你的 key。

"MiMo" 是小米的注册商标，mimo-tui 仅调用 MiMo Token Plan 的公开 API，与小米官方无任何关联。
